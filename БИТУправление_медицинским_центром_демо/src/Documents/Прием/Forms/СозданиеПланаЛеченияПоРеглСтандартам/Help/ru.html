Создание плана лечения по регламентированным стандартам 

План лечения

Таблица плана лечения содержит номенклатуру назначения и ее количество в плане лечения.


Таблица "НМУ" отображает НМУ, связанные с выбранным планом лечения.

Стандарты лечения


Таблица стандарта лечения содержит состав лечения из выбранного регламентированного стандарта. Колонки "Номенклатура" и "Кол-во в плане" соответствуют номенклатуре и количеству строки плана лечения, связанной с НМУ.

Добавить строки плана лечения для выделенных строк номенклатур медицинских услуг можно при помощи кнопки "Добавить в план". Также добавить строку плана можно двойным кликом по ячейке НМУ.

Связь выделенных строк номенклатур медицинских услуг с выделенным планом лечения создается либо кнопкой "Привязать к строке плана", либо командой "Привязать к строке плана" из контекстного меню, либо перетаскиванием строк таблицы на строки, с которыми желаем создать связь.

Кнопка "Добавить обязательные" открывает форму для создания связи с планами лечения для всех обязательных строк из состава лечения выбранного стандарта.
