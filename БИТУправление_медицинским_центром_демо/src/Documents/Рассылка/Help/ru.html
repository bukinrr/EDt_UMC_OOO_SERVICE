Документ "Рассылка SMS-сообщений"

Документ предназначен для документированной отправки sms-сообщений указанному списку получателей.

Документ доступен в подсистемах Продажи и Администрирование.

В документе следует указать обработку рассылки сообщений. Обработка рассылки получает от документа список получателей сообщений и в соответствии с заложенным в неё алгоритмом отправляет данные сообщения (как правило используя веб-сервис рассылки). В случае отправки сообщения, обработка отмечает строку получателей состоянием "Отправлено".

Список получателей может быть заполнен как вручную, так и командой заполнения из подменю "Заполнение получателей".

Для отправки сообщений следует воспользоваться командой "Отправить сообщения".

Сообщение может быть доставлено немедленно в момент отправки либо в указанное в документе время. Данные параметры передаются в обработку рассылки, которая должна их обрабатывать.

Формирование текста сообщения может быть выполнено либо путём указания единого текста, либо выбором Вида сообщения рассылки, по шаблону которого текст будет сформирован для каждого получателя индивидуально.
