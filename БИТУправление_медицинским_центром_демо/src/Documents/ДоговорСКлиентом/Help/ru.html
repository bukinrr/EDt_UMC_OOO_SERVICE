Документ "Договор с клиентом"

Документ предназначен для отражения факта подписания с клиентом формальных документов: договора на услуги, информированного согласия и других. Для документа обязательно указание клиента и вида договора. 
В случае, если для выбранного вида договора указан шаблон печати в формате MS Word, для договора доступна печать. В этом случае, в соответствии с шаблоном печати, будет сформирован и открыт для просмотра документ формата MS Word.

В случае, если для вида договора указан перечень номенклатуры услуг, на которую он распространяется, проведённый договор с клиентом также разрешает использовать в документах продажи эту номенклатуру из числа требующих наличия договора.
