План обмена "Обмен с филиалами"

Данный план обмена предназначен для организации обмена данными между узлами баз, работающих на этой конфигурации, для которых допускаются различия в конфигурациях, не затрагивающее данные, участвующие в обмене.

План обмена доступен для использования в случае, если в учетной политике включена опция  "Использовать обмен данными". В этом случае пользователю доступна подсистема "Обмен данными", где предоставляется возможность настройки нового обмена со сторонним узлом базы данных (добавление узла в план обмена), просмотра уже настроенных узлов обмена, выполнения обмена.
