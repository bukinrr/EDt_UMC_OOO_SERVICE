Контактная информация

Регистр предназначен для хранения контактной информации (телефон, адрес, e-mail, веб-страница и пр.) по клиентам, филиалам, сотрудникам.

Контактная информация хранится по видам, которые задаются в справочнике Виды контактной информации. Пользователь может задать произвольный набор видов контактной информации для каждого элемента соответствующего справочника.
