Консоль WorkList

В регистре "Консоль WorkList" хранится "Рабочий Лист Исследований" — список запланированных для пациентов исследований, который может быть получен запросом пользователя с DICOM-совместимого оборудования к РИС-системе. Регистр заполняется автоматически при планировании услуг, у которых заполнен реквизит "DICOM Modality", а в спецификации указано используемое оборудование.

При формировании "WorkList" ФИО пациента, наименование услуги, назначивший врач, исполнитель услуги и т.д. записываются  на латинице. Сведения о пациенте, как: пол, дата рождения, номер карты, а так же данные по услуге переводятся в формат, понятный стандарту DICOM 3. После выполнения исследования на аппарате в регистре при помощи службы MPPS  автоматически устанавливается отметка исполнения. Для случая, когда механизм MPPS не используется, в форме регистра есть возможность вручную поставить или снять отметку исполнения.

Если в спецификации услуги задан комплект оборудования, то при планировании данной услуги в форму регистра WorkList попадают несколько записей, соответствующих каждому аппарату отдельно. Услуга считается выполненной, если она выполнена на всех аппаратах заданного комплекта оборудования.

Разграничение доступа  к данным в форме списка записей регистра WorkList
В спецификации медицинской услуги задается комплект или список комплектов оборудования, на которых данная услуга выполняется.
Комплект может состоять из одного и более оборудований. Услуга считается (полностью) выполненной, если она выполнена на всех аппаратах комплекта.
Кроме того, услуга может выполняться либо на одном, либо на другом комплекте, тогда в спецификации задается список комплектов. В итоге, услуге может быть задан иерархический список оборудования, где в группах задаются комплекты.
Если услуга выполняется на нескольких аппаратах (комплекте оборудования), то в WorkList попадает несколько записей для каждого аппарата отдельно.
В настройках контроля исполнения в форме списка "Используемое оборудование" автоматически задается весь список оборудования по всем спецификациям услуг, которые выполняются данным врачом. Врач выбирает то оборудование, на котором он работает в текущий момент. Для работы на следующем аппарате врач выбирает требуемое оборудование. Данная настройка необходима для разграничения доступа  к данным в форме списка записей регистра WorkList, т.е. в WorkList попадают только те услуги, которые будут выполняться на выбранном врачом аппарате (или комплекте аппаратов).
