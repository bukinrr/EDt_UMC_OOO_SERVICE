Установка программы wkhtmltopdf для отправки приёмов в PDF

Для отправки приёмов в PDF используется программа wkhtmltopdf, распространяемая по лицензии GPL. Её необходимо установить на сервер (при работе в клиент-серверном варианте) или на локальную машину, при работе с файловой базой.

Windows

Загрузите и запустите установщик wkhtmltopdf. Прочитайте и согласитесь с лицензионным соглашением. Согласитесь с папкой установки по умолчанию (желательно).
После установки конвертер wkhtmltopdf будет доступен в любой папке (благодаря добавлению в переменную окружения %PATH% его папки).
Информацию по добавлению местоположений (папок) программ в системную переменную среды PATH смотрите в документации для вашей операционной системы.

Параметр "Расположение программы wkhtmltopdf" обязательно следует указать при клиент-серверном варианте базы, если у пользователя, под которым выполняется процесс сервера 1С, нет права использования утилиты командной строки. Указывается полный путь до исполняемого файла wkhtmltopdf на сервере (например, "C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe"). Также следует обеспечить право пользователя сервера на работу с каталогом установки wkhtmltopdf.

Linux

Для ОС Linux предпочтительней устанавливать wkhtmltopdf через сборку из исходных текстов. Ниже приведен перевод инструкции того, как выполняется такая установка. Оригинал инструкции был взят с официального сайта.

Перед компиляцией необходимо установить программы и библиотеки, от которых зависит конвертор: X11, gcc, git и openssl. На Debian/Ubuntu это можно сделать следующим образом:

sudo apt-get build-dep libqt4-gui libqt4-network libqt4-webkit
sudo apt-get install openssl build-essential xorg git-core git-doc libssl-dev

На других системах вам нужно использовать свой менеджер пакетов, в котором пакеты должны называться похожим образом.

Вначале необходимо получить модифицированную версию QT:

git clone git://gitorious.org/+wkhtml2pdf/qt/wkhtmltopdf-qt.git wkhtmltopdf-qt

Затем следует сконфигурировать, скомпилировать и установить QT. Отметим, что это займет некоторое время, в зависимости от того, какие аргументы используются для конфигурации QT.

cd wkhtmltopdf-qt
./configure -nomake tools,examples,demos,docs,translations -opensource -prefix ../wkqt
make -j3
make install
cd ..

Это все, что необходимо для компиляции программы wkhtmltopdf.

git clone git://github.com/antialize/wkhtmltopdf.git wkhtmltopdf
cd wkhtmltopdf
../wkqt/bin/qmake
make -j3

В результате должен сформироваться бинарный файл с именем wkhtmltopdf в текущей директории. Для нормальной работы необходимо сделать полную установку.

sudo make install

После установки станет доступна команда wkhtmltopdf, которая преобразует html-файл в pdf-файл.
