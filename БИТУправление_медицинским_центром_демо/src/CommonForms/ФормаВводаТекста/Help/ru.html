Форма ввода текста

Форма предназначена для ввода текстовых значений параметров медицинских документов.

На форме располагаются (сверху вниз):
1.     Название заполняемого параметра
2.     Поле для ввода текста
3.     Таблица готовыми фразами для данного параметра.

После ввода текста нажатие кнопки «Сохранить и закрыть» закрывает форму и отдает текст в исходный документ.
Кнопки «,», «.» и «;» - определяют знак пунктуации, которым готовая фраза при выборе будет отделена от уже введенного текста. Если все кнопки отжаты, она будет отделена пробелом.
Кнопка «Создать» дает возможность создать и сохранить новую готовую фразу для данного параметра.
Если для параметра, значение которого вводится, включен признак "Выбирать значение из заданного списка при использовании в шаблоне как ссылки", то будет также показана закладка "Значения ввода" с теми же значениями, что введены для параметра.
