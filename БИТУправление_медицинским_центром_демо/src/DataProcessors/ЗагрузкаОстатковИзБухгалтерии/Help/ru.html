Обработка "Загрузка остатков из 1С:Бухгалтерии предприятия"

Обработка загружает предварительно выгруженные из 1С.Бухгалтерии предприятия данные о номенклатуре и остатках. Для выгрузки следует использовать специальные правила выгрузки, входящие в дистрибутив программ и обработку универсального обмена XML - из конфигурации 1С либо с диска ИТС (раздел универсальных внешних обработок).

Для загрузки следует указать специализацию номенклатуры. которая будет создана при загрузке.
