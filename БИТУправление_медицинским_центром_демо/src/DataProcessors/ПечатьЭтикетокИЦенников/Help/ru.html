Обработка "Печать ценников и этикеток"

Для товаров можно распечатать ценники и этикетки товаров. Для печати этикеток товаров в справочнике Номенклатура для товара должен быть предварительно задан штрихкод товара.

Важно! При печати этикеток используется элемент управления 1С:Печать штрих-кодов. Для того чтобы печать этикеток была доступна, необходимо  установить элемент управления 1С:Печать штрих-кодов.

Ценники могут быть распечатаны по любому прейскуранту, определенному в справочнике "Прейскуранты".
