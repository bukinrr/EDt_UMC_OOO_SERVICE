Службы DICOM

Данная обработка предназначена для запуска служб "Storage SCP" и "MWL SCP", которые обеспечивают работу с DICOM-оборудованием.
Служба "Storage SCP" предназначен для получения файлов *.dcm (серий снимков или отдельных снимков) и сохранения их в каталоге, из которого пользователь (врач) может получить файлы для просмотра снимков в формате JPEG и прикрепления их к протоколам исследований. Может быть запущена либо на клиенте, либо на сервере.
Служба "MWL SCP" отвечает за передачу сведений о пациентах и их исследованиях (WorkLists) из МИС на медицинское оборудование.
