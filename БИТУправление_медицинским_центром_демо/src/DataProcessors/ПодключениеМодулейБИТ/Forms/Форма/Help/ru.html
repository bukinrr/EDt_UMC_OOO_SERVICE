Форма "Подключение модулей" предназначена для создания заявки на подключение выбранного модуля, которая будет отправлена на веб-сайт bit-medic. Для успешного создания заявки необходимо указать, ваше имя, телефон и адрес электронной почты в соответствующих полях настоящей формы. По нажатию кнопки "Создать заявку" конфигурация проинформирует пользователя об успешности создания данной заявки на веб-сайте. Данные передаются на веб-сайт bit-medic.ru по http протоколу, в случае необходимости можно перейти на веб-сайт в ручном режиме и создать заявку напрямую с сайта.
