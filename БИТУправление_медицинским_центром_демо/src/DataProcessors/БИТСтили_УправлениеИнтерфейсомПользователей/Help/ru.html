Оформление и настройка интерфейса

Возможности:

·        Выбор альтернативного цветового стиля или набора картинок подсистем.

·        Настройка расположения панелей и варианта интерфейса для профилей пользователей и общей - для пользователей, не имеющих такой настройки по профилю.


Оформление интерфейса


На закладке "Стиль оформления интерфейса" можно выбрать нестандартный цветовой стиль базы или набор картинок подсистем - из числа вариантов, заложенных в конфигурацию. А также вернуться к стандартным.

Оформление - общее для всех пользователей базы (не может быть индивидуальным).

Цвета и картинки выбираются независимо. Например, можно использовать цветовой стиль, а настройкой профилей вовсе скрыть отображение картинок. И наоборот, использовать альтернативные картинки со стандартными цветами платформы «1С:Предприятие 8.3».

    При использовании режима интерфейса Версия 8.2 цветовое оформление не будет работать!

    Пример: допустим, мы хотим установить цветовой стиль "Стерильный" с альтернативным черно-белым набором картинок. Для этого:

1).    На странице "Цветовое оформление" выбираем строку "Стерильный"

2).    Жмём кнопку "Применить". Отказываемся от перезапуска приложения.

3).    На странице "Картинки разделов" выбираем строку "чёрно-белые мелкие".

4).    Видим внизу в "Рекомендованные настройки набора", что разделы предлагается показывать слева, список окон снизу и показывать картинки разделов. Ничего не делаем с этим.

5).    Жмём кнопку "Применить набор". Соглашаемся перезапустить.

Итого: в базу установлены цветовой стиль, альтернативные картинки разделов, а пользователям создана общая настройка интерфейса - как в рекомендованных набора картинок.


Вернуть стандартные цвета/набор можно кнопками возврата на страницах стилей/наборов. Но не расположение панелей - оно настраивается или удаляется на странице "Настройки интерфейса пользователей".

При выборе строки стиля или набора справа показаны примеры внешнего вида, если они есть у выбранного стиля. Стрелки сверху - переключение между изображениями.

Под списком: описание стиля/набора, а также рекомендованные настройки интерфейса.

Если включен флаг "Сохранить настройки стиля/набора как общие настройки по умолчанию", то рекомендованные настройки установятся для всех пользователей, у которых нет настроек для профиля.

Примечание: кнопки "Применить" требуют, чтобы в этот момент не был открыт Конфигуратор базы с открытыми окнами расширений базы (т.е. не блокировал их).


Настройки интерфейса пользователей

Каждому пользователю можно указать профиль. Здесь для профиля можно установить настройки интерфейса (вариант, панели, отображение картинок разделов).

Также можно зафиксировать "общие настройки" - для тех пользователей, у кого нет профиля в карточке или для профиля нет настройки.

Если такой пользователь изменит себе настройку вручную (например, добавит еще одну панель), то при следующем запуске она отменится обратно в пользу настройки профиля (или общей).

Таким образом, например, индивидуальная настройка масштабирования из обработки "Управление настройками форм" будет действовать только если:

а).     Для этого пользователя нет настройки профиля (или общей настройки для всех);

б).    Настройка есть, но в ней вариант интерфейса "не задан".

Пример: хотим сделать общую настройку с разделами без картинок слева, а также настройку для профиля "Врачи со слабым зрением" с вариантом "Такси обычный", т.е. с возможностью масштабирования форм (увеличение до 400%).

1.       Общая настройка:

1.1.   Переключаем "Область применения настроек" на "Настройки общие"

1.2.   Ставим "Отображение панели разделов" в "Текст"

1.3.   Включаем "Задано расположение панелей", где Разделы - слева, Открытые – Низ.

1.4.   Жмём сохранить. Перезапускаем, проверяем.

 2.         Для врачей:

2.1.   Создаем профиль "Врачи со слабым зрением"

2.2.   Переключаем "Область применения настроек" на "Настройки профиля" и выбираем профиль.

2.3.   Ставим "Вариант интерфейса" в "Такси обычный" 

2.4.   Остальные настройки, например, ставим в состояние "Не задано".

2.5.   Жмём сохранить. Для проверки заходим под пользователем с этим профилем в карточке пользователя.

Для просмотра текущих настроек по строке нужен:

а).       двойной клик по строке (прочитанные настройки подставятся в поля на форме);

б).      выделить строку и нажать "Прочитать настройки".

Для варианта интерфейса "Версия 8.2" не применимо расположение панелей (их в нём нет). Эта настройка будет игнорироваться даже если сохранена.

Примечание: понятия «Вариант интерфейса», «Панели», «Отображение панели разделов» определяются платформой «1С:Предприятие 8.3» и подробно описаны её в руководстве пользователя. Визуализация может немного различаться у разных релизов платформы
