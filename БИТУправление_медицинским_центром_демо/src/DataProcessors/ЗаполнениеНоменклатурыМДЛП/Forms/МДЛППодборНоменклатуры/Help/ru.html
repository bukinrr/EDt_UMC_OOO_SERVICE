Форма предназначена для заполнения номенклатуры в уведомлениях МДЛП.
Сверху - список с позициями из документа, где номенклатура не заполнена.
Снизу - список номенклатур-кандидатов для текущей строки верхнего списка.
При открытии формы сразу производится поиск в текущей базе прошлых операций с этими GTIN. При нахождении номенклатура подставляется в верхнюю таблицу на правах самого достоверного источника.
Команда "Поиск по данным ЧЗ" через Интернет запрашивает в ЧЗ данные номенклатуры по GTIN. В случае получения идёт поиск подходящей номенклатуры в базе по реквизиту КЛП и по наименованию.
Если подходящей номенклатуры в базе нет, то номенклатуре может быть создана автоматически. Для этого нужно:
  - Выбрать в верхнем списке позиции для создания;
  - Указать специализацию для создаваемой номенклатуры;
  - Запросить данные из ЧЗ, если еще не были запрошены;
  - Запустить команду "Создать номенклатуру из ЕСКЛП".
Если по КЛП из ЧЗ удалось получить данные номенклатуры из ЕСКЛП (nsi.rosminzdrav.ru), то номенклатура будет создана.

Код уровня достоверности вариантов (способа поиска) подсказывает пользователю, какой вариант, возможно, требует перепроверки:

1 - номенклатура взята из прошлых операций МДЛП в этой базе (по GTIN)
2 - найдена по КЛП, полученному из ЧЗ по GTIN
3 - полученному из ЧЗ по GTIN
4 - создана в базе по данным ЧЗ и ЕСКЛП.
