Регистр накопления "Накопления по скидкам"

Регистр хранит обороты по накопительным скидкам для последующего анализа достижения очередного порога накопительной скидки.
Данные хранятся в разрезе получателя и вида скидки, таким образом, если действует несколько накопительных скидок, в регистр попадут записи по каждой из них в результате продажи.
В качестве регистраторов могут выступать следующие документы: "Оказание услуг", "Ввод начальных данных по накоплениям".
