Справочник "Клиенты"

Справочник "Клиенты" предназначен для хранения информации о клиентах.

При добавлении нового элемента справочника заполняются следующие поля:

  • Полное имя клиента;
  • Представление;
  • Группа;
  • Дата рождения;
  • Пол;
  • Источник информации, из которого клиент узнал об организации. 
  • Признак "Напоминать о дне рождения". Если установлен, то для данного клиента будут автоматически генерироваться задачи напоминания о дне рождения.

Табличная часть заполняется в несколько этапов:

  • На вкладке "Контактная информация" можно указать следующую информацию - телефон, адрес, e-mail, веб-страницу, номер ICQ;
  • Вкладка "Свойства". Здесь можно указать дополнительные свойства клиента;
  • На вкладке "Дополнительная информация" можно ввести любой дополнительный текст о клиенте. 
  • На вкладке "История посещений" можно посмотреть историю продаж услуг, товаров и сертификатов клиенту.
  • На вкладке "Даты клиента" можно описать особые даты клиента и указать вид события, которое будет автоматически генерироваться при приближении этой даты (в соответствии с настройками вида событий). Признак "Однократное" показывает, формировать задачу однократно при наступлении даты события или многократно ежегодно.
  • На вкладке "Штрихкоды" можно указать штрихкоды, по которым в дальнейшем можно быстро подбирать/находить клиента в документах и форме списка клиентов.
