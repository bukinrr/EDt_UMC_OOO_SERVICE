Справочник "Номенклатура"

Справочник "Номенклатура" хранит виды предоставляемых услуг и список материалов.

При добавлении нового элемента указывается:

  • наименование

  • полное наименование (подробное или официальное название)

  • вид номенклатуры (услуга, материал, набор услуг)

  • базовую единицу измерения из классификатора единиц измерения

  • категория выработки по этой номенклатуре

  • возможность учета по характеристикам (при условии. что в настройках учета установлен признак "Вести учет по характеристикам").

  • перечень дополнительных характеристик данной номенклатуры, при условии, что в этом элементе включена возможность учета по характеристикам

  • штрихкоды данной номенклатуры для подбора в документы с помощью сканера штрихкодов.

Для услуги можно указать также:

  • продолжительность - нормативная продолжительность работы. Это значение будет по-умолчанию подставляться в документ заявки

  • нормы расхода материалов. В дальнейшем в документе "Оказание услуг" можно будет заполнить табличную часть "Материалы" этими нормами (в пересчете на количество каждой услуги в документе) кнопкой "Заполнить по нормам" командной панели на закладке "Материалы".

  • оборудование из справочника «Оборудование», которое используется в при выполнении данной услуги

Для материала можно указать:

  • единицу хранения остатков, единицу товаров и дополнительные единицы из справочника "Единицы измерения". Единица хранения остатков обязательна к заполнению. В этой единице измерения хранятся остатки товаров в регистрах. Рекомендуется в качестве единицы остаток указывать минимальную единицу товара (штуки, граммы, миллилитры). Информация о количестве в отчетах выводится также в единице хранения остатков. Единица товаров не обязательна и служит для того, чтобы использоваться в качестве единицы по-умолчанию при выборе номенклатуры в документах поступления товаров и оказания услуг на закладке "Товары".

  • минимальные остатки номенклатуры - для филиала в целом или в разрезе отдельных складов. Эта информация будет в дальнейшем использоваться при построении отчета "Минимальные остатки товаров" для анализа запасов на складах.

 Для набора услуг можно указать:

  • Комплектующие набора - список услуг, комплектующих данный набор услуг.

Наборы услуг служат для быстрого подбора в документ пакета услуг. При подборе в документы набор разбивается на услуги, и они попадают в документ отдельными строками.
