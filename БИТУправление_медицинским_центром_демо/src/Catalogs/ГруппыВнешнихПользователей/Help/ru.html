Справочник используется для настройки групп внешних пользователей.

Группы создаются плоским списком или иерархическим. В иерархическом списке, внешние пользователи вложенных групп неявно входят в группы-родители.

Группы формируются по используемому функционалу прикладного решения:

  • Контрагенты
    • Доверенные партнеры
    • Основные покупатели
    • Основные поставщики
    • Участники опроса
  • Физические лица

Существует предопределенная группа "Все пользователи", которая неявно включает всех внешних пользователей.

Существует возможность создать группы внешних пользователей одного типа, в том числе такие группы, которые неявно включают всех пользователей одного типа.

Назначение групп внешних пользователей - это возможность указать в некоторых прикладных целях, не только внешнего пользователя, но и группу внешних пользователей.

Такой подход позволит осмысленно применять группы внешних пользователей различными пользователями и внешними пользователями информационной базы.

Если используется непосредственное назначение ролей, для внешних пользователей можно использовать назначение ролей через группы внешних пользователей.
