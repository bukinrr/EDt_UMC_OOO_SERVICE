Справочник "Виды состояний заявок"

Справочник "Виды состояний заявок" предназначен для ведения списка видов состояний документов "Заявка". Справочник содержит предопределенные ключевые состояния "Запланирована" и "Выполнена".

Для каждого состояния можно указать цвет фона заявок в этом состоянии при их просмотре в журнале записи. По-умолчанию в качестве цвета подставится значение, указанное в учетной политике как основной цвет заявки.

Тип состояния не обязателен к заполнению и может оставаться пустым. Если в заявке будет указано состояние с заполненным типом «Отменена» или «Выполнена», то такая заявка не будет отображаться в журнале записи. Поэтому для того, чтобы, например, заявки с состоянием "Выполнена" выводились в календарь планирования, тип состояния нужно оставить незаполненным.

Код элемента справочника влияет только на сортировку списка состояний в контекстном меню заявки в журнале записи и может редактироваться пользователем.

Использование состояний заявок регулируется параметром учетной политики «Использовать состояния заявок». В настройках учетной политики можно указать состояния заявок по-умолчанию: состояние новой заявки и состояние, которое будет автоматически устанавливаться в заявке при проведении документа "Оказание услуг", в котором она указана основанием.

Пользователь может добавлять свои виды состояний, например, для выделения важности заявки: "VIP-заявка".
